== Instruction

There is an model called global_config.rb and its test as global_config_spec.rb has corresponding code,
for better understand you read the test codes.

Please follow the following instruction:

= git clone https://github.com/rokibulhassan/ConfigurationSettings.git
* cd ConfigurationSettings

* <tt>rake db:create</tt>
* <tt>rake db:migrate</tt>
* <tt>rake doc:test:prepare</tt>
* <tt>rspec spec</tt>
